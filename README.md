# 📚 Fundamentos de HTML e CSS

Bem-vindo(a) ao meu repositório de estudos! 🚀

Este espaço foi criado para registrar minha jornada de aprendizado nos fundamentos de **HTML** e **CSS**, desenvolvendo a base necessária para criar páginas web modernas e bem estruturadas.

## 🎯 Objetivo

Aprender os principais conceitos de desenvolvimento web, praticando a criação de páginas com foco em:

- Estruturação de conteúdo com HTML
- Estilização de páginas com CSS
- Organização e boas práticas de código
- Construção de layouts responsivos
- Evolução contínua por meio da prática

## 📖 Conteúdos estudados

### HTML
- Estrutura básica de uma página
- Tags semânticas
- Títulos e parágrafos
- Links
- Imagens
- Listas
- Tabelas
- Formulários

### CSS
- Seletores
- Cores e tipografia
- Box Model
- Margin e Padding
- Display
- Flexbox
- Grid Layout
- Responsividade

## 💻 Tecnologias

- HTML5
- CSS3
- Visual Studio Code
- Git e GitHub
